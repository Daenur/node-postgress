Description NSI API

http://45.89.26.151:3001/api/gps/?lon=82&lat=19 - gps (Lon,Lat) - get location (country,state,district) by coords

(working with only India)

http://45.89.26.151:3001/api/states states - get states list

states (ids) - get states by id

states (name) - get states by name

http://45.89.26.151:3001/api/distr distr - get disgtricts

distr (ids) - get districts by state id

distr (idd) - get district by id

distr (mrd) - get district by MRD_code

distr (name) - get districts by name

distr (sname) - get districts by State name

http://45.89.26.151:3001/api/projects projects - get projects

project (idp) - get project by id

project (namep) - get project by name

http://45.89.26.151:3001/api/subprojects subprojects - get subprojects

project (idp) - get subprojects by id project

project (idsp) - get subproject by id subproject

project (namesp) - get subproject by name subproject

http://45.89.26.151:3001/api/polygon polygon - get polygon by ID

ids - get state polygon by id

idd - get dictricrt polygon by id

http://45.89.26.151:3001/api/polygon_lite - get lite version ( 300 point min) polygon by ID

ids - get state polygon by id

idd - get dictricrt polygon by id

http://45.89.26.151:3001/api/polygon_geojson polygon - get polygon by ID as GeoJSON

ids - get state polygon by id

idd - get dictricrt polygon by id

http://45.89.26.151:3001/api/polygon_geojson_lite - get lite version ( 300 point min) polygon by ID as GeoJSON

ids - get state polygon by id

idd - get dictricrt polygon by id

http://45.89.26.151:3001/api/polycenter polycenter - get polygon center point by ID

ids - get state polygon center by id

idd - get dictricrt polygon center by id

http://45.89.26.151:3001/api/childshapes/ shapechild - get child polygons by ID

ids - get child polygons by id state

names - get child polygons by name state
